# ADR-0003: 외부 데이터(Tier 2) 취급 정책 — 비영속 + 제공자 약관 준수 + 대가성 문구

- **상태**: 채택
- **일시**: 2026-06-18
- **관련**: phase M5, INV-4·INV-9·INV-10, ADR-0001, `docs/TECHNICAL` §1.2

## 맥락
Q1(상용 API 약관) 확인 결과, 가격·구매·리뷰를 공급하는 세 경로 모두 "실시간 표시·비영속" 모델이며 창고형 적재를 허용하지 않는다.

- **네이버 검색(쇼핑) API**: 일 25,000회 호출 제한. 검색결과의 별도 저장/DB화는 허용되지 않으며 캐시·출처표기 모델. 더하여 국내 판례상 **출처표기·링크를 달아도 공개데이터 재배포가 데이터베이스제작자권 침해로 인정**된 사례 다수(네이버부동산-다윈중개, 잡코리아-사람인). 이용자가 출처로 이동하지 않고 정보를 얻으면 트래픽·광고수익 감소를 이유로 침해 인정.
- **카카오 로컬 API**: 응답 결과 데이터를 별도로 저장하여 사용하는 것을 약관상 불허(개발자 FAQ 명시). 표시 + `place_url` 링크. 카카오맵 사용설정 필요(2024-12-01~) + 일/월 쿼터.
- **쿠팡 파트너스**: 제휴(딥링크) 모델. 2024-12-01 개정 공정위 「추천·보증 등에 관한 표시·광고 심사지침」상 **대가성 문구를 제목 또는 첫머리에 명확·확정형으로 표기 의무**(예: "이 게시물은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다"). '소정의 수수료를 지급받을 수 있음' 같은 조건부·불확정 표현 불가. 위반 시 수익 정산 제외·계정 정지 + 공정위 시정명령·과징금·형사처벌 가능.

## 결정
- ADR-0001의 Tier 2 비영속 원칙을 확정한다(INV-4). 세 경로 모두 request-time 조회 + 약관 허용 캐시(TTL) + 출처·조회시각 표기(INV-9)로만 사용한다.
- 가격 조회는 **상품 상세 진입 시 on-demand로만** 수행하고, 목록·스냅샷에는 가격을 싣지 않는다(네이버 일 25,000회·카카오 쿼터 관리).
- 쿠팡 파트너스 제휴 링크가 노출되는 모든 화면에 대가성 문구를 명확·확정형으로 눈에 띄는 위치(상품 영역 상단 또는 페이지 첫머리/사이드바 일괄고지)에 표기한다 — **INV-10**으로 강제.

## 근거
세 경로의 약관과 국내 DB제작자권 판례가 모두 같은 방향(비영속·표시·출처)을 가리킨다. 대가성 문구는 법적 의무이자 수익 보호 요건이다.

## 결과 / 트레이드오프
- 이득: 약관·DB권 리스크를 구조적으로 차단. 쿠팡 수익화는 정당 경로로 유지.
- 비용: 가격은 항상 실시간(지연·쿼터 소모). 카카오 장소는 저장 없이 매번 조회. 대가성 문구가 UI 레이아웃 제약이 됨.

## 검토한 대안
- 가격·장소를 주간 스냅샷에 적재 — 세 경로 약관 위반 + DB제작자권 침해 소지. 기각.
